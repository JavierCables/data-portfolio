Summary

Goal

The primary goal was to develop a robust Automatic Number Plate Recognition (ANPR) system using 100% Python. This involved accurately detecting moving license plates and reading the corresponding text in a video environment. A key aspect was combining object detection (YOLOv8) with object tracking (SORT) to assign unique IDs to vehicles throughout the video. This tracking was crucial for determining the single, most accurate license plate value for each car by selecting the read text with the highest OCR confidence score, thereby solving ambiguity issues across multiple frames. The project focused specifically on the United Kingdom license plate format.


Process

The process began with loading YOLOv8 models for car and license plate detection. Frames were read and vehicles detected. Object tracking (SORT) was implemented on vehicles to assign unique IDs. License plates were detected and assigned to their corresponding tracked cars using bounding box containment logic. The license plate image was cropped and pre-processed using grayscale conversion and thresholding to optimize OCR performance. EasyOCR read the text. Post-processing involved validating the text against the UK format and correcting common character confusions (e.g., S/5, O/0) for robustness. Results, including bounding box data and confidence scores, were structured by frame and car ID, saved to a CSV file, and interpolated for stable video visualization.


Insights

The project successfully implemented a complete end-to-end Automatic Number Plate Recognition system. By integrating deep learning object detection (YOLOv8) and robust object tracking (SORT), the solution could track vehicles and reliably associate license plate detections. Critical computer vision steps were applied, including image preprocessing (thresholding) to maximize OCR accuracy. Post-processing logic ensured that only texts matching the required format were accepted and provided automatic character correction to enhance reliability. The methodology leveraged the vehicle tracking data to select the single most confident license plate reading for each vehicle over time. The final output included a structured CSV file containing frame-by-frame vehicle and license plate data, ready for further analysis and video visualization.
